# Awesome CLI Tools 🛠️

A curated collection of useful command-line utilities and scripts.

## Installation

```bash
git clone https://github.com/awanawana/awesome-cli-tools.git
cd awesome-cli-tools
```

## Tools

- `git-quick` - Quick git commands
- `backup` - Simple backup script
- `monitor` - System monitor
